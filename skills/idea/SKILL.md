# Idea

Read all available context — the codebase, CLAUDE.md, any active plan files, recent conversation history, and the current state of the project. Then answer this single question:

**What's the single smartest, most radically innovative, accretive, useful, and compelling addition you could make at this point?**

Rules:
- One idea only. No lists. No hedging.
- Be specific and concrete — name files, features, integrations, architecture changes.
- Explain *why* it's the highest-leverage move right now given the current state of things.
- Keep it to 2-3 paragraphs max.
